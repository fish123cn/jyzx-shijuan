# 江义中学试卷资源库
本仓库用于存放 **广东省佛山市顺德区勒流江义中学** 学科试卷及配套答案解析，仅供校内学习、复习、教研使用。

---
## 人工智能技术使用说明
- ⚠️仓库内的试卷，**除非特殊说明，均为扫描件+AI提取题目**，**解答除非特殊说明，均为AI生成**，上传到仓库前会进行审查（但可能也会出问题，如果有问题，请提issues）。

## 📌 使用说明
1. 直接打开对应学科的 PDF 文件查看试卷
2. 带 `-答案` 后缀的文件为该科目参考答案
3. 所有文件通常为标准PDF格式，支持手机/电脑直接浏览
